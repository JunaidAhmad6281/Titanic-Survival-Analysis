# 🚢 Titanic Survival Analysis Dashboard

> An end-to-end Data Analytics project using **Python, Pandas, NumPy, Matplotlib, Seaborn, and Tableau** to analyze the famous Titanic dataset and build an interactive dashboard.

---

## 📌 Project Overview

This project explores the Titanic passenger dataset to identify the key factors affecting passenger survival.

The workflow includes:

- Data Cleaning
- Exploratory Data Analysis (EDA)
- Feature Analysis
- Data Visualization
- Interactive Tableau Dashboard
- Business Insights

The objective is to uncover meaningful patterns and present them through an easy-to-understand dashboard.

---



# 📂 Dataset

The dataset contains passenger information including:

- Passenger Class
- Age
- Gender
- Fare
- Embarked Port
- Family Size
- Survival Status

Target Variable:

- **Survived**
    - 0 = Not Survived
    - 1 = Survived

# 🔍 Key Insights

- Female passengers had significantly higher survival rates than males.
- Children showed better survival chances than adults.
- First-class passengers had the highest survival probability.
- Most passengers belonged to the age group between 20–40 years.
- Passenger class strongly influenced survival outcomes.
